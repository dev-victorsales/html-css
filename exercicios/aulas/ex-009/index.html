Trabalhando com listas

Listas ordenadas

As listas ordenadas estão entre <ol> e dentro de um <li> e elas aparecem como o exemplo a baixo.

  1. Acordar
  2. Ligar para o joão assim que acordar
  3. Tomar café
  4. Escovar os dentes
  5. Ir para Faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Podemos alterar a formatação de uma lista por um parametro type ou seja "<ol type = "Tipo aqui">

Podemos também escolher de onde começar usando "<ol type = ""> start= "onde começar (1-infinito)."

Listas não ordenadas

São aquelas onde a sequencia não precisa ser numerada. Apenas demarcada, utilizamos o comando <ul> para uma lista não ordenada. Elas aparecem como o exemplo a baixo

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Podemos alterar a formatação de uma lista por um parametro type ou seja "<ul type = "Tipo aqui">
